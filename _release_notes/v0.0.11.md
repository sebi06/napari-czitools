# napari-czitools v0.0.11

Release date: 2026-04-10

## Highlights

- Dropped Python 3.11 support — minimum is now Python 3.12 (aligned with czitools).
- Bug fixes in `_io.py` and `_reader.py`.

## What Changed

### Python version support

- Dropped Python 3.10 and 3.11; `requires-python` is now `>=3.12, <3.14`.
- Updated classifiers in `pyproject.toml` to list only 3.12 and 3.13.
- Updated `tox.ini` matrix to `py{312,313}`.
- Updated CI workflow matrix to `["3.12", "3.13"]`.
- Updated documentation (`README.md`, `.github/copilot-instructions.md`) to reflect the new minimum version.

### Bug fixes

- Fixed issues in `src/napari_czitools/_io.py` related to CZI file reading.
- Fixed issues in `src/napari_czitools/_reader.py` for improved reader stability.

### Housekeeping

- Renamed `RELEASE_NOTES/` folder to `_release_notes/`.

## Notes

- The Python 3.11 drop follows the upstream decision in czitools.
- CI/PyPI publication is triggered from tags matching `v*`.
- This release is created from `main`.
