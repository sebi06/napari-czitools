# napari-czitools v0.0.10

Release date: 2026-03-29

## Highlights

- Replaced the custom hand-coded `DoubleRangeSlider` (~600 lines) with thin wrappers around **superqt**'s `QLabeledRangeSlider` and `QRangeSlider`.
- Fixed single-frame extraction: both slider handles can now sit on the same value (e.g. 4–4).
- Updated CI workflow actions to Node.js 24-compatible versions.

## What Changed

### Slider refactoring

- Rewrote `src/napari_czitools/_doublerange_slider.py`:
  - `LabeledDoubleRangeSliderWidget` now wraps superqt's `QLabeledRangeSlider`.
  - `DoubleRangeSlider` now wraps superqt's `QRangeSlider`.
  - Added `_allow_handle_overlap()` helper to enable single-frame extraction (both handles at same value).
  - Full type annotations added.
  - Public API unchanged — all consumers work without modification.
- Rewrote `src/napari_czitools/_tests/test_doublerange_slider.py` (31 tests covering wrapper API).

### CI workflow updates (Node.js 24)

- `actions/checkout` v4 → v6 (Node.js 24)
- `actions/setup-python` v5 → v6 (Node.js 24)
- `codecov/codecov-action` v3 → v5 (Node.js 20, replaces deprecated Node.js 16)

### Documentation

- Updated `README.md` with superqt slider descriptions and compatibility notes.
- Updated `.github/copilot-instructions.md` with slider refactoring guidance and handle-overlap patch documentation.

## Validation

- Full test suite run in the `smartmic` environment:
  - `python -m pytest -v`
  - Result: `54 passed, 16 warnings`
